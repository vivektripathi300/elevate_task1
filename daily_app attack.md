Daily-used applications have multiple attack surfaces such as web apps, mobile apps, APIs, networks, and cloud, which attackers exploit to steal data or gain unauthorized access.

Mapping Daily-Used Applications to Attack Surfaces
1. Email Applications (Gmail, Outlook)

Attack Surfaces:

Web application (email web login)

Mobile application

APIs (email syncing, attachments)

Network (SMTP, IMAP, POP3)

Possible Attacks:

Phishing emails

Malware attachments

Account takeover

Example (Exam Line):
Email apps are vulnerable to phishing and credential theft through web and API attack surfaces.

2. WhatsApp / Messaging Apps

Attack Surfaces:

Mobile application

APIs (message delivery, media sharing)

Network (internet communication)

Cloud backup storage

Possible Attacks:

Malicious links

Account hijacking

Spyware through attachments

Example (Exam Line):
Messaging apps face risks through mobile apps, APIs, and cloud storage attack surfaces.

3. Banking Applications

Attack Surfaces:

Mobile application

Web application (net banking)

APIs (payment, balance check)

Network (internet, ATM networks)

Cloud infrastructure

Possible Attacks:

Phishing scams

Man-in-the-Middle attacks

Credential theft

Malware

Example (Exam Line):
Banking apps are exposed to web, mobile, API, and network attack surfaces, making them high-value targets.
